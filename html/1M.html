About 1M

한국산업기술대학교 디자인공학부 공모전 제출을 위해 기획하고 디자인한 반응형 웹사이트입니다.
디자이너 3명으로 구성되었고 기획부터 디자인에 전부 참여하였습니다.
React.js를 사용해 OPEN API를 가져와 검색 페이지를 구현하였고 TypeScript 학습 후 변환 예정입니다.

사용 기능

  • Frgma
  • API
