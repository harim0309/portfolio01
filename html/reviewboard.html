About ReviewBoard

SK Planet에서 진행한 ASAC 프로그램 기간 중 기업 프로젝트에 참여해서 제작한 웹사이트입니다.
‘엘핀’ 에서 제공한 기획을 바탕으로 UI를 수정하고 React로 구현했습니다.
프론트앤드 2명, 백앤드 1명으로 구성되었고 그 중 프론트앤드 팀에 속해 작업했습니다.
프론트엔드 Kakao Map API를 사용해 구현한 부분을 제외한 나머지 부분을 구현했습니다.

사용 기능

  • React.JS
  • Redux
  • styled-components
  • react-intersection-observer
  • AXIOS
  • Restful API
  • SCSS
  • 반응형
  • Kakao Map API
  • KakaoTalk Social Login API
  • Git hub
  • Swagger UI
